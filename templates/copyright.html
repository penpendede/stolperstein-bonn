Software

Die hier verwendete Software ist Open Source und kann auf GitHub eingesehen werden. Dort finden Sie die Lizenz der eigentlichen Software wie auch die Lizenzen der verwendeten Bibliotheken und Hilfsprogramme.

Stolpersteine

Die Daten der Stolpersteine werden mit Hilfe der Overpass-API direkt aus der Datenbank von OpenStreetMap entnommen.

Für die Daten von OpenStreetMap gilt die Open Database License (ODbL) v1.0

Fotos

Die Internetaddressen der angezeigten Fotos werden den von OpenStreetMap gelieferten Daten entnommen. Zu den Lizenzen der einzelnen Fotos klicken Sie bitte auf das jeweilige Foto; Sie werden dann – sofern möglich – zur entsprechenden Internetseite bei Wikimedia Commons weitergeleitet.

Grenzen der Stadt, Stadtbezirke und Ortsteile

Die verwendeten Informationen sind von der Stadt Bonn als OpenData der Allgemeinheit zur Verfügung gestellt worden.

  • Lizenz: creativecommons.org/publicdomain/zero/1.0/
  • Ortsteile: opendata.bonn.de/dataset/flächen-der-ortsteile
  • Stadtbezirke: opendata.bonn.de/dataset/flächen-der-stadtbezirke
  • Stadt: opendata.bonn.de/dataset/fläche-stadtgebiet-bonn
